Tema ovog rada je Online prodavnica. Ime prodavnice u ovom slucaju je "S Store".

Program je napravljen da prikazuje artikle u ponudi prodavnice, daje opciju da ih ubacite i izbacite iz vase korpe i da završite kupovinu pritiskom na "CHECKOUT" dugme.
Takodje, u ovom programu se nalaze i precice koje vode na odredjene delove sajta kao i precica za korpu.

Za izradu ovog sajta su korišceni HTML, CSS, CSS Grid Layout i Java Script.

Struktura se sastoji iz:
Osnovne strane:
	Header-a koji sadrži precice za korpu i pojedine delove sajta;
	Samog sadržaja web stranice, koja u sebi ima 4 dela (Notebooks, Smartphones, Smartwatches i Infomation). U prva tri dela postoji tekst koji opisuje kategoriju i prozvode u ponudi vezane za tu kategoriju. Dok se u INFORMATION nalaze kontakt informacije.
Stranice CART, u kojoj su proizvodi koji se trenutno nalaze u korpi, opcija za CHECKOUT i ukupna vrednost trenutne korpe;
CSS fajlova za izgled obe stranice;
JavaScript fajla koji sadrži informacije o svim proizvodima u ponudi i zajednicke funkcije;
